ADRs follow Michael Nygard's format. New ADRs:

1. Copy `0001-multi-cloud-strategy.md` as a template.
2. Number sequentially.
3. Open a PR; review by the platform team and at least one consumer-team representative.
4. Status starts as `Proposed`. Becomes `Accepted` on merge, `Superseded` when replaced.
